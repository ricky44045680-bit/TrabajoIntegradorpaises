## Descripción
Aplicación en Python para gestionar datos de países (nombre, población, superficie, continente) con persistencia en archivo CSV.

## Funcionalidades
- Agregar país
- Actualizar población/superficie
- Búsqueda por nombre
- Filtros múltiples
- Ordenamiento ascendente/descendente
- Estadísticas completas

## Estructura del proyecto
- main.py → código principal
- paises.csv → base de datos inicial (20 países)
- README.md → este archivo

## Cómo ejecutar
```bash
python main.py
